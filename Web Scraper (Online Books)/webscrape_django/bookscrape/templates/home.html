Welcome to the E-Book Scraper tool

The is a web-scraping tool that enable users scrape book information from the 'Books To Scrape' website. The 'Get Book Lists' function allows users to retrieve a list of books on all pages under a specific topic/genre while the 'Book Price' function enables users to get price information of a specific book based on the topic and book input.

This tool was developed as a sample project using Python's Django Web Framework.
